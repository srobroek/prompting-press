# prompting-press (TypeScript / Node)

The npm distribution of Prompting Press — a thin wrapper over the Rust napi-rs
binding crate [`prompting-press-node`](../../crates/prompting-press-node). Built
with `@napi-rs/cli`.

**Skeleton only (spec 001 / FR-004):** no runtime code ships yet. `src/generated/`
holds the schema-derived TypeScript types (code-generated — **do not edit**;
regenerated by `pnpm codegen` and freshness-gated in CI). `"private": true` until
spec 007 wires publishing.

The thin loader over the compiled `.node` addon lands with the binding (spec 005).
