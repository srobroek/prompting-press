### `schemas/scripts/codegen-check.sh` (new)

Shell script that:
1. Calls `git add -N` on the three generated paths (catches untracked newly-created files)
2. Loops over all three paths and collects failures
3. On any failure: prints named list of drifted files + full `git diff` + actionable message, exits 1
4. On clean: prints PASSED with file list, exits 0
